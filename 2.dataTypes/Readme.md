# data types

## primitive

- interge ==> byte, short,int, long
- float ==> double, float
- character
- boolean

#### integer

- byte ==> 1 byte ==> -2^7 to 2^7 -1 ==> -128 to 127
- short ==> 2 bytes
- int ==> 4bytes
- long ==> 8bytes

#### float

- float ==> 4 bytes
- double(default in java) ==> 8 bytes

- e.g.
  ```java
  double num =  5.6;
  float num =  5.6F;
  ```

#### character

- char ==> 2 bytes

  ```java
    char c = 'k';
  ```

- for string double quotes `""` , for character single quotes `''`

#### boolean

- `true` or `false`
- `true` and `false` are keywords
